def range(start : Int, end : Int, step : Int) : Range
Create an iterator with elements en+1 = en + step where e0 = start and ei < end. Will return an empty range for nonsensical range/step arguments.
Parameters
start - the start value of the iterator
end - the end value of the iterator
step - the increment value of the iterator (must be positive or negative)
Returns
the iterator with values in range [start;end).
